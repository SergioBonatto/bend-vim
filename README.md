### On VIM
Clone this repo and past the file "bend.vim" into `~/.vim/syntax/bend.vim` and add in your `.vimrc` file the next line:

```vim
au BufRead,BufNewFile *.bend set filetype=bend
au BufNewFile,BufRead *.bend set syntax  =bend
```
